# Paisa-N-Market 📈

Paisa-N-Market is a comprehensive financial application built with Next.js. It offers tools for budgeting, personalized financial advice, portfolio tracking, and fraud prevention to help users manage their finances effectively.

## 🚀 Features

- **Dashboard:** A central hub for all your financial overviews.
- **Budgeting & Expense Tracking:** Manage your budgets and track daily expenses.
- **Portfolio Management:** Keep track of your investments and assets.
- **Personalized Financial Advice:** Get tailored advice and recommendations.
- **Fraud Prevention:** Learn to protect yourself with fraud challenges and resources.

## 🛠️ Tech Stack

- **Framework:** [Next.js](https://nextjs.org/) (App Router)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **UI Components:** [Radix UI](https://www.radix-ui.com/)
- **Backend/Platform:** [Firebase](https://firebase.google.com/)
- **AI Integration:** [Genkit](https://firebase.google.com/docs/genkit) (Google AI)

## 📦 Getting Started

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/AnkitaChoudharyN/Paisa-N-Market.git
   cd Paisa-N-Market
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:9002](http://localhost:9002) in your browser to see the result.

## 📜 Available Scripts

- `npm run dev` - Starts the Next.js development server on port 9002.
- `npm run build` - Builds the application for production.
- `npm start` - Starts the production server.
- `npm run genkit:dev` - Starts the Genkit AI environment.
